# **App Name**: Barasho Hub

## Core Features:

- Centralized Dashboard: Dashboard with sections for school management, registration, finance, exams, and reports.
- Student Enrollment Form: Form to enroll new students with personal and academic details.
- Student Directory: Display, search, and filter registered students.
- Financial Tracking: Input and track school finances.
- Exam Management: Schedule exams and input results.
- Report Generation: Generate summary reports for different parts of the application
- Role Management: Role-based access control.

## Style Guidelines:

- Primary color: Saturated blue (#29ABE2) to convey trust and professionalism.
- Background color: Light gray (#F0F2F5) for a clean and modern interface, in light mode.
- Accent color: Soft teal (#70C5B5) for interactive elements and highlights.
- Body and headline font: 'Inter', a sans-serif font, for a modern, machined, objective look, suitable for a dashboard.
- Use flat, minimalist icons to represent different modules and actions.
- Intuitive layout with clear visual hierarchy. Use charts for data visualization.
- Subtle animations for transitions and feedback.
